# 择日学

择日学是选择有利时间进行重要活动的学问，核心是"趋吉避凶，顺天应时"。

---

## 一、基本概念

### 为什么要择日

- 时间本身携带五行能量（年月日时各有干支）
- 特定时间的能量与人的八字、事件性质相配合，可增加成功概率
- 避开冲克、凶煞，减少阻碍

### 择日的层次

```
黄历择日（民间通用）
    ↓
八字择日（结合当事人命盘）
    ↓
奇门择日（高级，结合奇门遁甲）
    ↓
玄空择日（结合玄空飞星）
```

## 二、黄历（通书）择日

### 黄历的基本结构

黄历每天标注：
- 当日干支
- 当日宜忌
- 冲煞方位
- 吉神凶煞

### 十二建星（十二值日）

每日对应一个建星，循环排列：

| 建星 | 含义 | 适合 |
|------|------|------|
| 建 | 建立、强健 | 祭祀、出行 |
| 除 | 除旧、清除 | 扫舍、治病 |
| 满 | 圆满、充实 | 婚嫁、开业 |
| 平 | 平稳、平和 | 修造、出行 |
| 定 | 稳定、确定 | 签约、安床 |
| 执 | 执行、固执 | 捕猎、纳财 |
| 破 | 破坏、冲破 | 诸事不宜（凶日） |
| 危 | 危险、高处 | 祭祀（忌登高） |
| 成 | 成功、完成 | 开业、婚嫁、动土 |
| 收 | 收获、收藏 | 纳财、入仓 |
| 开 | 开启、开放 | 开市、出行、婚嫁 |
| 闭 | 封闭、结束 | 安葬、塞穴 |

**吉日**：成、开、定、执（次吉）
**凶日**：破日最凶，危日次之

### 常见黄历宜忌事项

| 事项 | 宜选日 | 忌选日 |
|------|--------|--------|
| 婚嫁 | 成日、开日、满日 | 破日、冲本命日 |
| 开业 | 成日、开日 | 破日、闭日 |
| 动土 | 成日、开日 | 破日、冲太岁 |
| 安葬 | 闭日 | 重丧日 |
| 出行 | 建日、平日 | 破日、往亡日 |
| 签约 | 定日 | 破日 |

### 常见凶煞

| 凶煞 | 含义 |
|------|------|
| 岁破 | 与太岁相冲的月份，诸事不宜动土 |
| 月破 | 与月建相冲之日，凶 |
| 往亡 | 出行不吉 |
| 重丧 | 安葬忌用，主重复丧事 |
| 杨公忌 | 民间传说的13个凶日 |
| 四离四绝 | 节气前一天，诸事不宜 |

### 杨公十三忌

正月十三、二月十一、三月初九、四月初七、五月初五、六月初三、七月初一、七月廿九、八月廿七、九月廿五、十月廿三、十一月廿一、十二月十九

## 三、八字择日

### 基本原则

结合当事人（或主事人）的生辰八字，选择与其命盘相配合的日期。

**核心要求**：
1. 所选日柱不冲克当事人的年柱、日柱
2. 所选日的五行能补充当事人命盘所需
3. 所选日与事件性质相符

**具体规则**：
- 婚嫁：所选日不冲新郎新娘的年支、日支
- 动土：所选日不冲房主的年支
- 开业：所选日五行旺财星

### 年家奇门择日

结合奇门遁甲，选择吉门、吉星、吉神所在方位对应的时间出行或行事。

---

## 四、奇门择日（奇门遁甲择时）

### 基本思路

排出当前时间的奇门盘，找到吉门（开、休、生）与吉星（天辅、天心、天任）叠加的宫位，在该方位行事或朝该方向出发。

### 常用吉格

| 格局 | 组合 | 含义 |
|------|------|------|
| 青龙返首 | 值符+开门 | 大吉，万事顺遂 |
| 飞鸟跌穴 | 值使+生门 | 求财大吉 |
| 天遁 | 天芮+生门+太阴 | 隐匿、保护 |
| 地遁 | 天柱+休门+九地 | 隐匿 |
| 人遁 | 天心+开门+九天 | 出行大吉 |

### 时家奇门 vs 日家奇门

- **时家奇门**：以时辰为单位排盘，精度高，用于具体行动时间
- **日家奇门**：以日为单位排盘，用于选日

## 五、重要活动择日要点

### 婚嫁择日

1. 不冲新人年支（生肖相冲）
2. 不冲新人日支（日柱地支）
3. 避开破日、往亡日
4. 月份：避开新人的"冲月"
5. 常用吉日：成日、开日、满日

**生肖六冲**（婚嫁忌冲）：
```
鼠↔马  牛↔羊  虎↔猴  兔↔鸡  龙↔狗  蛇↔猪
```

### 开业择日

1. 选成日、开日
2. 时辰选生门、开门时辰
3. 五行配合：行业五行旺的时间
   - 金融/珠宝（金）：选金旺之日时
   - 餐饮（火）：选火旺之日时
   - 木材/植物（木）：选木旺之日时

### 动土/装修择日

1. 不冲房主年支
2. 不冲太岁（当年地支）
3. 避开土煞方位动土

### 出行择日

1. 选建日、平日
2. 奇门选吉门方位出发
3. 避开往亡日、月破日

## 七、择日与其他玄学的关系

```
命理（八字）→ 了解个人喜忌五行 → 指导择日方向
风水（玄空）→ 了解当前元运旺衰 → 配合择日
奇门遁甲   → 提供精确时空吉凶 → 精准择时择方
黄历通书   → 提供基础宜忌参考 → 日常择日依据
```

择日是玄学各分支的综合应用，好的择日师需要融会贯通命理、风水、奇门等多个体系。
