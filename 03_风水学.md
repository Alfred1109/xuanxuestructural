# 风水学

风水学（堪舆学）是研究环境与人的关系的学问，核心是"藏风聚气"——寻找或营造有利于人居住、生活的环境。

> "风水之法，得水为上，藏风次之。" ——《葬书》郭璞

---

## 一、基本概念

### 气

风水的核心是"气"：
- **生气**：有利于人的能量，需要聚集
- **煞气**：不利于人的能量，需要化解
- **气乘风则散，界水则止**：气遇风会散，遇水会停聚

### 龙、穴、砂、水、向（风水五诀）

| 要素 | 含义 |
|------|------|
| 龙 | 山脉走势，来龙去脉 |
| 穴 | 气聚之处，建筑或墓地的核心位置 |
| 砂 | 穴位四周的山丘，起护卫作用 |
| 水 | 水流方向与形态 |
| 向 | 建筑朝向 |

## 二、形势派（峦头派）

以山川地形、自然环境为主要判断依据，注重实地观察。

### 四神兽（四灵）

```
        朱雀（南/前）
玄武（北/后）  ←  穴位  →  白虎（西/右）
        青龙（东/左）
```

- **玄武**（后）：靠山，要高耸有力
- **朱雀**（前）：明堂，要开阔平坦
- **青龙**（左）：左砂，要蜿蜒有情
- **白虎**（右）：右砂，要低伏顺从

口诀：**"左青龙，右白虎，前朱雀，后玄武"**

### 理想地形

```
      ███ 玄武（靠山）███
    ██                  ██
   █  青龙（左）  白虎（右）█
    ██                  ██
      ░░░ 明堂（前方平坦）░░░
         ～～ 水流 ～～
```

- 背山面水
- 左高右低（青龙高于白虎）
- 前有案山（近处低山）、朝山（远处高山）
- 水流弯曲环抱（玉带水）

### 砂的形态

**吉砂**：
- 旗砂（旗帜形）：出武将
- 笔砂（毛笔形）：出文人
- 印砂（方正形）：出官员

**凶砂**：
- 探头砂：有小人
- 反弓砂：不聚财
- 割脚砂：主伤病

### 水的形态

**吉水**：
- 玉带水（弯曲环抱）
- 聚水（水聚明堂）

**凶水**：
- 反弓水（水流背离）
- 直射水（直冲而来）
- 割脚水（水流紧贴建筑）

## 三、理气派

在形势基础上，加入方位、时间等理气计算，更为精细。

### 八宅派

以房屋坐向将住宅分为**东四宅**和**西四宅**，人也分为**东四命**和**西四命**，人宅相配则吉。

**八宅方位吉凶**：

| 宅/命 | 生气 | 天医 | 延年 | 伏位 | 绝命 | 五鬼 | 六煞 | 祸害 |
|-------|------|------|------|------|------|------|------|------|
| 吉凶 | 大吉 | 吉 | 吉 | 小吉 | 大凶 | 凶 | 凶 | 小凶 |

**东四宅**：坎宅（坐北朝南）、离宅、震宅、巽宅
**西四宅**：乾宅、坤宅、艮宅、兑宅

**东四命**：坎命、离命、震命、巽命（出生年尾数对应）
**西四命**：乾命、坤命、艮命、兑命

## 四、阳宅风水要点

### 外部环境

1. **道路**：避免T字路口正冲、高架桥压顶
2. **建筑**：避免尖角对冲（壁刀煞）
3. **水体**：前有水（聚财），避免反弓水
4. **地形**：避免低洼地、山谷风口

### 内部布局

| 区域 | 风水要点 |
|------|---------|
| 大门 | 纳气之口，宜宽敞明亮，避免正对电梯、楼梯 |
| 客厅 | 宜方正，沙发背靠实墙 |
| 主卧 | 床头靠实墙，避免横梁压床 |
| 厨房 | 灶台避免正对门、窗 |
| 卫生间 | 不宜在房屋中央，门不对床 |

### 常见煞气与化解

| 煞气 | 成因 | 化解方法 |
|------|------|---------|
| 壁刀煞 | 邻楼尖角对冲 | 凸镜、山海镇 |
| 穿心煞 | 大门对后门 | 屏风、玄关 |
| 天斩煞 | 两楼间缝隙对冲 | 葫芦、铜钱剑 |
| 横梁压顶 | 梁压床/沙发/书桌 | 竹节装饰遮盖 |
| 镜子煞 | 镜子正对床 | 移走或遮盖 |

## 五、阴宅风水（墓地）

阴宅风水认为祖先安葬之地的气场会影响后代，核心仍是"藏风聚气"。

- 选址要求与阳宅类似：背山面水，四神护卫
- 穴位要"藏风聚气"，土质要"暖而不燥，润而不湿"
- 朝向结合后代生辰八字选择
