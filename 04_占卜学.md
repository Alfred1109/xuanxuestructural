# 占卜学

占卜学是通过特定方法，在特定时间对特定事件进行预测的学问。核心思想是"时间信息包含事件信息"。

---

## 一、六爻纳甲（文王卦）

### 基本概念

用铜钱或蓍草起卦，得到六爻卦象，通过纳甲（将天干地支纳入卦中）来分析事情的吉凶走向。

### 起卦方法

**铜钱摇卦**：
- 三枚铜钱，摇六次，每次记录结果
- 三正（字）= 老阴（变爻）：× 
- 两正一反 = 少阴：- -
- 两反一正 = 少阳：—
- 三反（花）= 老阳（变爻）：○

从下往上排列六爻，得到本卦；老阴老阳变爻后得到变卦。

### 纳甲规则

六十四卦每卦六爻，各爻纳入天干地支：

**乾宫八卦**（以乾卦为例）：
```
上爻：壬戌（土）
五爻：壬申（金）
四爻：壬午（火）
三爻：甲辰（土）
二爻：甲寅（木）
初爻：甲子（水）
```

### 六亲

以卦宫五行为基准，确定六亲：
- **父母爻**：生我者
- **兄弟爻**：同我者
- **子孙爻**：我生者
- **妻财爻**：我克者
- **官鬼爻**：克我者

### 六神

按起卦日的天干，将六神（青龙、朱雀、勾陈、腾蛇、白虎、玄武）从初爻到上爻依次安入。

### 分析要点

1. **用神**：所问之事对应的爻（问财看妻财爻，问官职看官鬼爻）
2. **世爻**：代表问卦者自身
3. **应爻**：代表对方或事情结果
4. **动爻**：老阴老阳为动爻，是变化的关键
5. **月建、日辰**：对各爻有生克冲合的影响

**旺相休囚死**：
- 旺：与月建同五行
- 相：月建生之
- 休：生月建
- 囚：克月建
- 死：月建克之

## 二、梅花易数

### 基本概念

北宋邵雍所创，以先天八卦数为基础，通过起卦时的时间、数字、方位、声音等信息起卦，快速判断吉凶。

### 起卦方法

**时间起卦**：
- 上卦 = （年数 + 月数 + 日数）÷ 8 取余数
- 下卦 = （年数 + 月数 + 日数 + 时数）÷ 8 取余数
- 动爻 = （年数 + 月数 + 日数 + 时数）÷ 6 取余数

**数字起卦**：见到任何数字，按上述方法计算

**物象起卦**：见到某物、听到某声，取其先天数起卦

### 先天八卦数

```
乾=1  兑=2  离=3  震=4
巽=5  坎=6  艮=7  坤=8
```

### 分析方法

1. **体卦**：包含世爻的卦（上卦或下卦），代表问卦者
2. **用卦**：另一卦，代表所问之事
3. **体生用**：我付出，结果一般
4. **用生体**：对我有利，吉
5. **体克用**：我能控制局面，吉
6. **用克体**：对我不利，凶
7. **互卦**：二三四爻组成下互卦，三四五爻组成上互卦，代表事情中间过程
8. **变卦**：动爻变化后的卦，代表最终结果

### 物象断卦

梅花易数重视"象"的运用：
- 乾：天、圆、金、父、君、头
- 坤：地、方、土、母、腹、布
- 震：雷、动、木、长男、足、竹
- 巽：风、木、长女、股、绳
- 坎：水、险、中男、耳、血
- 离：火、日、中女、目、文书
- 艮：山、止、少男、手、石
- 兑：泽、金、少女、口、刀

## 三、奇门遁甲

### 基本概念

奇门遁甲是中国最高级的预测与应用体系，集天文、历法、哲学、军事于一体，被称为"帝王之学"。

**三奇**：乙（日奇）、丙（月奇）、丁（星奇）
**六仪**：戊、己、庚、辛、壬、癸
**八门**：休、生、伤、杜、景、死、惊、开
**九星**：天蓬、天芮、天冲、天辅、天禽、天心、天柱、天任、天英
**八神**：值符、腾蛇、太阴、六合、白虎（勾陈）、玄武、九地、九天

### 奇门盘结构

```
┌──────┬──────┬──────┐
│ 巽宫 │ 离宫 │ 坤宫 │
│ 东南 │  南  │ 西南 │
├──────┼──────┼──────┤
│ 震宫 │ 中宫 │ 兑宫 │
│  东  │  中  │  西  │
├──────┼──────┼──────┤
│ 艮宫 │ 坎宫 │ 乾宫 │
│ 东北 │  北  │ 西北 │
└──────┴──────┴──────┘
```

每宫包含：天干（地盘+天盘）、八门、九星、八神

### 阴遁阳遁

- **阳遁**：冬至后，六仪顺布，三奇逆布（夏至前）
- **阴遁**：夏至后，六仪逆布，三奇顺布（冬至前）

共 8（节气）× 3（局）× 2（阴阳）= 480局（实用540局）

### 奇门应用

1. **预测**：判断事情吉凶走向
2. **择时**：选择有利时间行事（出行、谈判、求职）
3. **择方**：选择有利方位出发
4. **布局**：在特定方位布置物品以改变气场
5. **符咒**：结合道家符咒使用（高级应用）

### 吉门吉星速记

**吉门**：开门（万事开头）、休门（休养生息）、生门（求财求生）
**凶门**：死门、惊门、伤门（次凶）、杜门（封闭）、景门（虚火）

**吉星**：天辅、天心、天任、天英（次吉）
**凶星**：天蓬、天芮（最凶）、天柱、天冲（次凶）

## 四、太乙神数

太乙神数是"三式"（奇门、六壬、太乙）之首，以太乙星为核心，主要用于预测国家大事、战争、自然灾害等宏观事件，民间应用较少。

- 以16宫为基础（不同于九宫）
- 计算太乙星、计神、客主等的位置
- 主要用于国运、战争预测

---

## 五、六壬神课（大六壬）

### 基本概念

六壬是"三式"之一，以十二地支为基础，通过月将（太阳所在宫次）和占时，推算四课三传，判断事情吉凶。

### 基本结构

**四课**：
```
第四课  第三课  第二课  第一课
（干上神）（支上神）（干）  （支）
```

**三传**：从四课中提取初传、中传、末传，代表事情的开始、经过、结果。

### 十二天将

| 天将 | 对应地支 | 含义 |
|------|---------|------|
| 贵人 | 丑/未 | 贵人相助 |
| 螣蛇 | 巳 | 惊恐、虚惊 |
| 朱雀 | 午 | 文书、口舌 |
| 六合 | 卯 | 和合、婚姻 |
| 勾陈 | 辰 | 田土、纠纷 |
| 青龙 | 寅 | 财喜、吉庆 |
| 天空 | 戌 | 空亡、欺骗 |
| 白虎 | 申 | 凶丧、疾病 |
| 太常 | 未 | 衣食、宴会 |
| 玄武 | 亥 | 盗贼、阴私 |
| 太阴 | 酉 | 阴私、女性 |
| 天后 | 子 | 女性、阴事 |

## 六、占卜体系对比

| 体系 | 难度 | 起卦方式 | 擅长领域 | 特点 |
|------|------|---------|---------|------|
| 梅花易数 | ★★☆ | 时间/数字/物象 | 日常小事 | 简便快捷，重象 |
| 六爻纳甲 | ★★★ | 铜钱摇卦 | 具体事件 | 精准，有固定规则 |
| 奇门遁甲 | ★★★★ | 时间排盘 | 预测+应用 | 可预测可布局 |
| 大六壬 | ★★★★★ | 月将+占时 | 综合预测 | 最精密，学习难度最高 |
| 太乙神数 | ★★★★★ | 固定推算 | 国家大事 | 宏观预测 |
