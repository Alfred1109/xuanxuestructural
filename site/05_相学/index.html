    相学¶

    相学是通过观察人的外貌特征（面部、手部、体型等）来判断人的性格、命运、健康的学问。

    "相由心生，境由心造" ——相学核心理念

    一、面相学¶

    基本框架¶

    三停（面部纵向三等分）：

    发际线 ─────────────── 上停（天庭）：主早年运（0-30岁）
         额头
眉毛   ─────────────── 中停（人中）：主中年运（30-60岁）
         鼻子
人中   ─────────────── 下停（地阁）：主晚年运（60岁后）
         嘴巴、下巴

    五岳： - 东岳（左颧）、西岳（右颧）、南岳（额头）、北岳（下巴）、中岳（鼻子）

    四渎： - 耳（江）、眼（河）、口（淮）、鼻（济）

    十二宫¶

    宫位 位置 主看
    命宫 两眉之间（印堂） 整体命运、精神状态
    财帛宫 鼻子 财运、理财能力
    兄弟宫 两眉 兄弟关系、朋友缘
    田宅宫 上眼皮（眼睑） 不动产、家庭
    男女宫 眼下卧蚕 子女缘、异性缘
    奴仆宫 下巴两侧 下属、晚年运
    妻妾宫 眼尾（鱼尾） 婚姻、配偶
    疾厄宫 鼻梁（年寿） 健康状况
    迁移宫 额角（驿马位） 出行、变动
    官禄宫 额头中央（中正） 事业、官运
    福德宫 额角（天仓） 福气、祖德
    父母宫 额头两侧（日月角） 父母缘、长辈缘

    五官详解¶

    额头（天庭）： - 宽广饱满：聪明、早年顺遂 - 低窄：早年辛苦 - 有横纹：操劳、波折

    眉毛： - 眉长过眼：兄弟多，人缘好 - 眉浓：个性强，有魄力 - 眉散乱：兄弟缘薄，性格不稳 - 眉头相连（连心眉）：多思虑

    眼睛： - 眼神有光：精神好，运势佳 - 三角眼：心机重 - 丹凤眼：聪明，贵气 - 眼白多（三白眼）：性格偏激

    鼻子（财帛宫）： - 鼻梁挺直：财运稳定 - 鼻头圆润（准头丰）：聚财 - 鼻孔外露：漏财 - 鼻梁有节：中年波折

    嘴巴： - 嘴角上扬：乐观，晚年好 - 嘴唇厚：重感情，口福好 - 嘴唇薄：说话直接，感情淡 - 嘴角下垂：晚年辛苦

    耳朵： - 耳廓厚大：长寿，福气好 - 耳朵贴头：做事踏实 - 耳垂肥厚：富贵相 - 耳位高（高于眉）：聪明

    下巴（地阁）： - 下巴圆润饱满：晚年好，有福气 - 下巴尖：晚年孤独 - 下巴宽：晚年稳定

    痣的含义（简述）¶

    • 面部痣：位置不同含义不同
    • 红痣（朱砂痣）：吉痣
    • 黑痣：需结合位置判断
    • 眉中痣：聪明，有贵气
    • 眼角痣：感情丰富，多桃花
    • 鼻头痣：财运有阻
    • 嘴角痣：口福好，善言

    二、手相学¶

    手的基本形态¶

    手型分类： | 手型 | 特征 | 性格 | |------|------|------| | 方形手 | 手掌方正，手指方 | 务实、稳重 | | 圆锥形手 | 手指尖细 | 艺术感强、感性 | | 匙形手 | 指尖宽扁 | 实干、创造力强 | | 哲学形手 | 指节明显 | 思考型、哲学气质 |

    三大主线¶

            食指  中指  无名指  小指
          |    |     |      |
    ╭─────────────────────────╮
    │  ①生命线（大鱼际弧线）   │
    │  ②智慧线（横穿手掌）     │
    │  ③感情线（上方横线）     │
    ╰─────────────────────────╯
          拇指侧        小指侧

    生命线（围绕大鱼际）： - 代表：体质、生命力、重大变化 - 长而清晰：体质好，生命力强 - 短或断裂：体质较弱或有重大变故 - 注意：生命线长短≠寿命长短

    智慧线（从食指与拇指间横穿）： - 代表：思维方式、智力、判断力 - 长而清晰：思维清晰，理性 - 向下弯曲：想象力丰富，感性 - 分叉：多才多艺

    感情线（小指下方横线）： - 代表：感情状态、感情观 - 长而清晰：感情丰富，重感情 - 短：感情较淡 - 多分叉：感情经历丰富

    辅助线¶

    • 命运线（纵向中线）：事业运、人生方向
    • 太阳线（无名指下纵线）：名誉、成就感
    • 婚姻线（小指下横线）：婚姻状况
    • 财运线：小指下斜线，财运
    • 健康线：斜穿手掌，健康状况

    丘（手掌隆起部位）¶

    丘 位置 含义
    木星丘 食指下 野心、领导力
    土星丘 中指下 责任感、命运
    太阳丘 无名指下 艺术、名誉
    水星丘 小指下 商业、沟通
    金星丘 拇指根部 爱情、活力
    月丘 手掌外侧下方 想象力、直觉
    火星丘 手掌中央 勇气、意志力

    三、其他相学¶

    体相¶

    • 骨相：骨骼形态，主贵贱
    • 气色：面部气色变化，主近期运势
    • 红润：运势好，健康
    • 发暗：运势受阻
    • 发黄：财运有阻，脾胃问题
    • 发青：惊吓、肝胆问题

    声音相¶

    • 声音洪亮：中气足，有领导力
    • 声音清亮：聪明，贵气
    • 声音嘶哑：运势受阻
    • 笑声爽朗：性格开朗，人缘好

    步态相¶

    • 步伐稳健：做事踏实
    • 步伐轻盈：思维灵活
    • 走路外八：性格外向
    • 走路内八：性格内敛
