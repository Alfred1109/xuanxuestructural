## 一、阴阳学说

### 基本概念

阴阳是描述宇宙万物对立统一关系的哲学模型。

| 阳 | 阴 |
|----|----|
| 天 | 地 |
| 日 | 月 |
| 火 | 水 |
| 动 | 静 |
| 刚 | 柔 |
| 男 | 女 |
| 明 | 暗 |
| 热 | 寒 |

### 四大法则

1. **对立**：阴阳相互对立，如寒热、明暗
2. **互根**：阴阳互为根本，孤阴不生，独阳不长
3. **消长**：阴阳处于动态平衡，此消彼长
4. **转化**：阴阳在一定条件下可以相互转化，"物极必反"

### 太极图解读

```
    ╭──────╮
   ╱  阳(白) ╲
  │  ●(阴鱼眼) │
  │            │
  │  ○(阳鱼眼) │
   ╲  阴(黑) ╱
    ╰──────╯
```

- 白鱼中有黑点：阳中有阴
- 黑鱼中有白点：阴中有阳
- 整体圆形：循环往复，无始无终

## 二、五行学说

### 五行属性

| 五行 | 方位 | 季节 | 颜色 | 数字 | 脏腑 | 情志 |
|------|------|------|------|------|------|------|
| 木 | 东 | 春 | 青绿 | 3、8 | 肝胆 | 怒 |
| 火 | 南 | 夏 | 红 | 2、7 | 心小肠 | 喜 |
| 土 | 中 | 长夏 | 黄 | 5、10 | 脾胃 | 思 |
| 金 | 西 | 秋 | 白 | 4、9 | 肺大肠 | 悲 |
| 水 | 北 | 冬 | 黑 | 1、6 | 肾膀胱 | 恐 |

### 五行生克关系

**相生**（促进、滋养）：
```
木 → 火 → 土 → 金 → 水 → 木（循环）
```
- 木生火：木燃烧生火
- 火生土：火燃烧成灰土
- 土生金：土中藏矿金
- 金生水：金属熔化似水，或金属遇冷凝水
- 水生木：水滋润树木生长

**相克**（制约、克制）：
```
木 → 土 → 水 → 火 → 金 → 木（循环）
```
- 木克土：树根穿透土壤
- 土克水：土能堤坝挡水
- 水克火：水能灭火
- 火克金：火能熔化金属
- 金克木：金属刀斧砍木

**相乘**（过度克制，病态）：克制太过
**相侮**（反克，病态）：被克者反过来克制克者

### 五行与数字

河图数：
- 天一生水，地六成之
- 地二生火，天七成之
- 天三生木，地八成之
- 地四生金，天九成之
- 天五生土，地十成之

### 爻的基本概念

- **阳爻**：— （一横，奇数9）
- **阴爻**：- - （两横，偶数6）
- 从下往上数：初爻、二爻、三爻、四爻、五爻、上爻
- **九五之尊**：第五爻为阳爻，是最尊贵的位置

### 易经核心思想

1. **变易**：宇宙万物时刻在变化
2. **不易**：变化中有不变的规律
3. **简易**：复杂现象背后有简单规律

## 五、河图洛书

### 河图

```
      7
   2     6
8    5    4
   3     9
      1
```
- 奇数（1,3,5,7,9）为阳，白点
- 偶数（2,4,6,8,10）为阴，黑点
- 五行方位由此而来

### 洛书（九宫格）

```
4  9  2
3  5  7
8  1  6
```
- 横竖斜三数之和均为15
- 是奇门遁甲、紫微斗数等的空间模型基础
